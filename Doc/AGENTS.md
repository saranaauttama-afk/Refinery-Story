# Refinery Story - AI Rules

- Build gameplay first, not architecture.
- Keep scope small.
- Do not refactor unrelated files.
- Every change must keep the game runnable.
- Use React + TypeScript + Vite.
- Use simple hardcoded data first.
- No ECS, no complex engine, no backend, no multiplayer.
- Current goal: make a Kairosoft-lite refinery management game.
