# Refinery Story - Game Design

## Genre

Management Simulation

Inspired by:

* Game Dev Story
* Dungeon Village
* Other Kairosoft games

---

## Core Loop

Buy Crude
→ Refine Fuel
→ Complete Contracts
→ Earn Money
→ Earn Research
→ Upgrade
→ Expand
→ Repeat

---

## Resources

### Money

Used for:

* Buildings
* Upgrades
* Expansion

### Crude Oil

Input resource.

### Gasoline

Primary output resource.

### Research Points

Used to unlock technologies.

---

## Buildings

### Crude Tank

Purpose:

* Increase crude storage

### Distillation Unit

Purpose:

* Increase production rate

### Product Tank

Purpose:

* Increase gasoline storage

---

## Combo System

### Crude Tank + Distillation Unit

Bonus:
+10% Production Rate

### Distillation Unit + Product Tank

Bonus:
+10% Sell Price

### Crude Tank + Product Tank

Bonus:
+10% Storage

---

## Design Principles

Players should always have:

* A short-term goal
* A medium-term goal
* A long-term goal

The game should reward optimization.

The game should avoid unnecessary complexity.

Fun is more important than realism.
