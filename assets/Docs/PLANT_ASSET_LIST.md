## Pixel Rules

Use these extra rules whenever the model starts drifting away from true pixel art:

1. Add `true pixel art sprite, strict pixel grid, nearest-neighbor look, no anti-aliasing, no painted shading`
2. Add `real transparent alpha background only, no fake transparency checkerboard`
3. Add `no soft studio backdrop, no blurred shadow field, no render-style background`
4. If the generator keeps faking transparency, generate on a flat single-color background first, then remove the background in a second step
5. If the generator keeps smoothing edges, ask for `low-resolution sprite upscale, crisp pixel edges, no subpixel smoothing`
